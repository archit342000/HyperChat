1. Core Architecture (The "Shell")

Scrolling Strategy (Mandatory): Native Window Scroll.

body must apply min-h-screen and overflow-x-hidden.

Do NOT use overflow-hidden on the body or h-screen on the main content wrapper.

Logic: Let the browser handle the viewport height to prevent mobile address bar bugs.

Layout Mode:

Mobile (< 768px): flex-col (Header sticky top, content stacked below).

Desktop (â‰¥ 768px): flex-row (Sidebar fixed left, content fluid right).

Z-Index Layering (Strict):

Base: z-0 (Ambient Backgrounds).

Content: z-10 (Main Scroll View).

Sidebar/Header: z-40 (Sticky elements).

Overlays (Modals/Drawers): z-[100].

Tooltips/Toasts: z-[110].

A. Desktop Sidebar ("Navigation Rail")

Position: sticky top-0 h-screen.

Visual Spec:

Light: bg-white border-r border-violet-100.

Dark: bg-[#121212] border-r border-white/5.

Texture: Opaque, but with a subtle bg-gradient-to-b from white to violet-50/30.

Dimensions & Resizing:

Default: Width w-64 (256px).

Collapsed: Width w-20 (80px).

Resizable Range: Min 240px to Max 480px (when expanded).

Transition: width 400ms cubic-bezier(0.2, 0.8, 0.2, 1) (Spring). Disable transition during manual drag.

Behavior:

Collapsible: Via toggle button or dragging width below threshold.

Resizable: via Drag Handle on right edge.

Drag Handle:

Hit Area: 4px wide transparent strip on right border.

Visual: On hover/active, show 2px vertical line bg-violet-300 (Light) / bg-violet-700 (Dark).

Cursor: col-resize.

Text Handling: When collapsed, text labels must set opacity: 0, width: 0, and display: none to prevent layout shifts.

Active State:

Indicator: 3px rounded vertical bar on the left (bg-violet-600 rounded-r-full).

Background: Tint bg-violet-50 (Light) / bg-violet-900/20 (Dark).

Icon: Color text-violet-700 (Light) / text-violet-300 (Dark).

B. Mobile Drawer

Type: Fixed Overlay (fixed inset-0).

Trigger: Hamburger button in md:hidden header.

Animation: Slide in from left with Ease-Out-Quint.

Backdrop: bg-violet-900/20 + backdrop-blur-sm. Clicking the backdrop MUST close the drawer.

C. Content Constraints

Max Width: Main content container MUST NOT exceed max-w-7xl (approx 1280px) for a slightly airier feel.

Centering: mx-auto (Centered horizontally).

Padding: px-6 (Mobile) / px-12 (Desktop).

3. Surface & Spacing System

The "Luminous Surface" (Card)

Concept: Physical matter that interacts with colored light.

Visual Spec:

Light: bg-white, border-violet-100, shadow-[0_4px_20px_-2px_rgba(124,58,237,0.08)] (Colored Shadow).

Dark: bg-[#1E1E1E], border-white/5, shadow-none.

Radius: rounded-2xl (16px) - Friendlier, softer corners.

Padding: Generous p-8 (32px).

Interaction: On hover, translate-y-[-2px] and shadow intensifies.

Spacing Tokens (Base-4)

Gap Small: 8px (gap-2).

Gap Medium: 24px (gap-6).

Gap Large: 48px (gap-12).

Section Spacing: 96px (gap-24).

4. Color System

Primary Accent: Electric Violet (#7C3AED / violet-600).

Secondary Accent: Soft Slate (#94A3B8 / slate-400).

Background:

Light: Gradient to-br from-violet-50 via-white to-slate-50.

Dark: Neutral-950 (#0A0A0A).

Text:

Primary: Slate-900 (Light) / Gray-50 (Dark).

Muted: Slate-500 (Light) / Gray-400 (Dark).

Validation:

Error: Rose-500 text/border. Background bg-rose-50.

Success: Teal-500 text/border. Background bg-teal-50.

6. Asset Guidelines

Photography:

Subject: High-quality workspace setups, abstract architecture, nature macro.

Style: Bright, airy, slightly desaturated but with preserved contrast.

Radius: rounded-2xl strictly.

Abstracts: Fluid shapes, glass refraction, soft mesh gradients (Violet/Teal/Pink).

7. Design Pattern Language

The Physics: "Coated Paper & Light". Elements are matte surfaces that catch colored ambient light.

Opacity: Surfaces are opaque, but shadows are colored.

Lighting: Diffused, soft area lighting. No harsh spotlights.

8. Extension Protocol (How to Build New Components)

If you need to build a component not listed above:

Container: div with bg-white, border-slate-100, shadow-sm, rounded-2xl.

Title: Manrope Bold.

Data: JetBrains Mono or Inter Tabular.

Action: Violet Gradient Button.

Color: Use Violet for primary, Slate for structure.

Spacing: Use multiples of 8px (padding 24px/32px preferred).

9. Motion

Curve: cubic-bezier(0.2, 0.8, 0.2, 1) (Spring-like, bouncy).

Durations:

Hover: 200ms.

Layout/Collapse: 400ms.

Modal Entry: 300ms.
